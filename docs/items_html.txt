<h2><span class="mw-headline" id="Notes">Notes</span></h2>
<ul><li><span class="template_sbc"><b>Soul Siphon</b></span> deals <span class="glossary" style="white-space:pre; position:relative;" data-game="lol" data-tip="Default damage"><span typeof="mw:File"><a href="/en-us/Damage" title="Damage"><img src="/en-us/images/thumb/Nocturne_Umbra_Blades.png/20px-Nocturne_Umbra_Blades.png?61439" decoding="async" loading="lazy" width="20" height="20" class="mw-file-element" srcset="/en-us/images/thumb/Nocturne_Umbra_Blades.png/40px-Nocturne_Umbra_Blades.png?61439 2x" data-file-width="64" data-file-height="64" /></a></span> <a href="/en-us/Damage" title="Damage">default damage</a></span>, and thus will not trigger <a href="/en-us/Spell_effects" class="mw-redirect" title="Spell effects">spell effects</a>.</li>
<li><span class="template_sbc"><b>Soul Siphon's</b></span> damage is blocked by <span class="glossary" style="white-space:pre; position:relative;" data-game="lol" data-tip="Spell shield"><span typeof="mw:File"><a href="/en-us/Spell_shield" title="Spell shield"><img src="/en-us/images/thumb/Sivir_Spell_Shield.png/20px-Sivir_Spell_Shield.png?4c4d1" decoding="async" loading="lazy" width="20" height="20" class="mw-file-element" srcset="/en-us/images/thumb/Sivir_Spell_Shield.png/40px-Sivir_Spell_Shield.png?4c4d1 2x" data-file-width="64" data-file-height="64" /></a></span> <a href="/en-us/Spell_shield" title="Spell shield">spell shield</a></span> upon the missile hitting the spell shield user.</li>
<li><span class="template_sbc"><b>Soul Siphon</b></span> is an <a href="/en-us/Auto-targeted" class="mw-redirect" title="Auto-targeted">auto-targeted</a> effect.
<ul><li>The healing missile has a global range.</li></ul></li>
<li>Triggering <span class="template_sbc"><b>Soul Siphon</b></span> launches a <span class="glossary" style="white-space:pre; position:relative;" data-game="lol" data-tip="Projectile"><span typeof="mw:File"><a href="/en-us/Projectile" title="Projectile"><img src="/en-us/images/thumb/Projectile.png/20px-Projectile.png?af216" decoding="async" loading="lazy" width="20" height="20" class="mw-file-element" srcset="/en-us/images/thumb/Projectile.png/40px-Projectile.png?af216 2x" data-file-width="64" data-file-height="64" /></a></span> <a href="/en-us/Projectile" title="Projectile">projectile</a></span> to each of the affected units, despite the effect being instant.
<ul><li>The healing missile and damaging missile land after 0.<small>32</small> seconds and 0.<small>4</small> seconds, respectively.</li></ul></li>
<li><span class="template_sbc"><b>Soul Siphon</b></span> will not trigger from healing an ally that is at full health or from healing instances under 20.</li>
<li><span class="template_sbc"><b>Soul Siphon</b></span> affects <span class="glossary" style="white-space:pre; position:relative;" data-game="lol" data-tip="Untargetable"><span typeof="mw:File"><a href="/en-us/Untargetable" title="Untargetable"><img src="/en-us/images/thumb/Untargetable_icon.png/20px-Untargetable_icon.png?c022c" decoding="async" loading="lazy" width="20" height="20" class="mw-file-element" srcset="/en-us/images/thumb/Untargetable_icon.png/40px-Untargetable_icon.png?c022c 2x" data-file-width="64" data-file-height="64" /></a></span> <a href="/en-us/Untargetable" class="mw-redirect" title="Untargetable">untargetable</a></span> allies.</li>
<li><span class="champion-ability-icon"><span class="inline-image label-after champion-icon" style="display:inline;white-space:pre;" data-param="" data-champion="Mel" data-skin="Original" data-game="lol"><span class="border" style="display:inline-block;position:relative;" data-width="20"><span class="mw-image-border" typeof="mw:File"><a href="/en-us/Mel" title="An icon representing Mel"><img alt="An icon representing Mel" src="/en-us/images/thumb/Mel_OriginalSquare.png/20px-Mel_OriginalSquare.png?dbac6" decoding="async" loading="lazy" width="20" height="20" class="mw-file-element" srcset="/en-us/images/thumb/Mel_OriginalSquare.png/40px-Mel_OriginalSquare.png?dbac6 2x" data-file-width="128" data-file-height="128" /></a></span></span> <span style="white-space:normal;"><a href="/en-us/Mel" title="Mel">Mel's</a></span></span> <span class="inline-image label-after ability-icon" style="display:inline;white-space:pre;" data-param="" data-champion="Mel" data-ability="Rebuttal" data-game="lol"><span class="border" style="display:inline-block;position:relative;" data-width="20"><span class="mw-image-border" typeof="mw:File"><a href="/en-us/Mel#Rebuttal" title="An icon for Mel&#39;s ability Rebuttal"><img alt="An icon for Mel&#39;s ability Rebuttal" src="/en-us/images/thumb/Mel_Rebuttal.png/20px-Mel_Rebuttal.png?b5846" decoding="async" loading="lazy" width="20" height="20" class="mw-file-element" srcset="/en-us/images/thumb/Mel_Rebuttal.png/40px-Mel_Rebuttal.png?b5846 2x" data-file-width="64" data-file-height="64" /></a></span></span> <span style="white-space:normal;"><a href="/en-us/Mel#Rebuttal" title="Mel">Rebuttal</a></span></span></span> may occasionally fail to attribute <span class="template_sbc"><b>Soul Siphon's</b></span> damage projectile to herself.<sup><span class="basic-tooltip" style="border-bottom:1px dotted gray;cursor:help;" title="This is not intended.">(bug)</span></sup>
<ul><li>This can cause the enemy to be able to kill their own allies, and gain all the kill's <a href="/en-us/Kill#Rewards" title="Kill">rewards</a>.<div style="clear:left"></div></li></ul></li></ul>
